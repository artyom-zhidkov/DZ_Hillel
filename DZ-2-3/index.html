task 1. Написать скрипт, который посчитает, сколько будет 2^10 с помощью цикла.

task 2. Написать скрипт, который запрашивает 2 числа, начало и конец диапазона, и в результате выдает сумму всех нечетных чисел в этом диапазоне.

task 3. Пользователь вводит диапазон нужно вывести на экран все простые числа из этого диапазона. Простыми числами называются числа, которые имеют всего 2 делителя, то есть делятся только на 1 и на самого себя.

task 4. Дано произвольное целое число n. Написать программу, которая разбивает число n на цифры и выводит их на экран; подсчитывает сколько цифр в числе n; находит сумму цифр числа n; меняет порядок цифр числа n на обратный.

task 5. Дан массив: [4, 23, 7, 39, 19, 0, 9, 14, 45, 93, 117, 55, 38, 24, 87] Найти сумму элементов массива в пределе [20, 80] Отсортировать массив (Метод вставок).

task 6. Дана строка: Я занимаюсь фигурным катанием с 5 лет. Очень хорошо помню этот день,1 сентября, когда мама привела в спортивную школу. Моя первая победа была в 8 лет. А сколько было падений? Их уже не перечесть. Огромное спасибо моему тренеру, который помогал и верил в меня! Найти сумму всех чисел в строке.
